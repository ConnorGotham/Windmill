Configuration

The Register Name is used to identify register values returned from the eGauge. Totaling Register is a read-only property expressing if the data point is a virtual, computed value. The Type determines where to use the accumulated value or the rate-of-change, instantaneous value in acquiring new values for the datapoint. The Multiplier is a scaling factor applied to new values. The Offset is added to new values. Not that both of these are applied after the conversion to kWh, if the data source has been configured to perform this conversion.
