PakBus Discovery Tool Help

The PakBus Discovery Tool is useful to configure dataloggers and points as well as explore the tables of a datalogger.

Configuration

  • The Logger Address is the address of the datalogger to connect to.
  • The Neighbor Address is a known neighbor address for the datalogger, leave as 0 if no neighbor is known.
  • The Round trip time (ms) is expected duration of a round trip request. Too small a value may result in lost comms due to too many retries.
  • The Security code is the datalogger's security code, leave as 0 if no code is required.
  • Table request retry delay is used in the data source to retry failed table configuration requests.

Useage

Once the above information is properly entered in the fields click the Connect to logger button. This will attempt to connect to the datalogger using the above settings as well as the data source specific connection settings from the data source panel. After a successful connection the tool will request the table definitions from the datalogger and report that the tool is ready.

Logger Tables Panel

When the logger table definitions are ready a panel with all the tables is shown. By clicking on the play button for a table the column definitions are shown for that table.

Logger Table Panel

When viewing a single table the columns and thier types are displayed. It is possible to request the latest values by clicking the play button at the top of the table. To add a column as a data point click the play button to the right of the desired column.

Setting Logger Time

The datalogger's current time can be requested by clicking the Get time button. To set the time of the logger, enter a valid time into the Datalogger time input and click the Set time button. To set the datalogger to the Mango server's current time click the Set to server time button. Note that the time input will reference datalogger local time.

File Explorer

The files for a datalogger can be displayed by clicking the Request files button. When the file list has been retrieved the datalogger files panel will be shown. The current running program will be displayed with a gear icon in the Running column. All other files will have a gear with green plus icon that can be clicked to set that file as the current running program. The green down arrow downloads the file.
