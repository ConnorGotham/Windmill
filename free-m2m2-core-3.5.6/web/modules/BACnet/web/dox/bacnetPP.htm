The Network number is the number of the network in which the device for the BACnet object is found. A value of 0 indicates the local network. The MAC is the id of the device within the network. Link service is the MAC of the device that acts as a router to other networks if this device for the object is foreign to the local network. If the device is on the local network, this value can be empty. The Device instance number is the BACnet inter-network unique identifier of the remote device.

The Object type is the type of object from which values will be requested. The Object instance number is the unique identifier of the object/type from which values will be requested. Points request the present value of an object instance.

The Use COV subscription determines whether Mango will attempt to subscribe to COV events for the object on the device. Not all objects support COV subscriptions, and not all devices support them even on objects that normally would. Moreover, some devices limit the amount of concurrent COV subscriptions they will accept, so no guarantees can be made that COV reporting will be established at runtime.

Use Settable to determine whether the interface will allow users to enter value with which to update the object. Only "output" values are considered to be settable.

The Write priority determines the priority to use when sending a write command to the object.

The Data Type field determines what Mango data type the point will have. Values returned from the BACnet network will automatically be converted to the required type.
