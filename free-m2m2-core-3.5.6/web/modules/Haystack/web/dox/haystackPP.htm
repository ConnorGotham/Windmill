Configuration for Haystack points.

  • id - String ID of the tag to poll/watch
  • settable - NOT CURRENTLY IMPLEMENTED

Data type coercion is supported for the following:

Mango Alphanumeric

  • Haystack Boolean
  • Haystack Numeric
  • Haystack String

Mango Binary

  • Haystack Boolean
  • Haystack Numeric (1,0)
  • Haystack String (true/anything else)

Mango Multistate

  • Haystack Boolean (true=1, false=0)
  • Haystack Numeric (0,1,2,...)
  • Haystack String ("1","2",...)

Mango Numeric

  • Haystack Boolean (true = 1.0, false = 0.0)
  • Haystack Numeric (0.0,1.0,...)
  • Haystack String ("1.0","2.0",...)
