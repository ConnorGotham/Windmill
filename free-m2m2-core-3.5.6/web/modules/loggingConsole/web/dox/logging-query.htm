RQL Queries for Logfiles

This terminal allows performing queries on the ma.log file. The query syntax is RQL. Always use a reasonable limit to avoid memory issues in the browser or Mango. Wildcards are regex style .* for match all. RQL requires that if using a limit command it must be the last command in the query.

Query-able Columns

  • level - Logging level
  • classname - Java Class name
  • method - Java method name
  • time - Timestamp for log
  • message - String logging message

Query Examples

  • level=INFO&limit(10)
  • (level=INFO|level=DEBUG)&limit(10)
  • like(classname,.*m2m2.*)&limit(10)
