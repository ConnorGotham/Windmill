Publisher Overview

The Twilio publisher is used to send messages through the Twilio service to your phone.

Twilio publishers can be useful in a number of ways:

  • Post a singular piece of data to any phone number.
  • Send out collections of data formatted into a given string.

The publisher configuration requires you to have a Twilio account. It requires you to know the Twilio URL you post to, your Twilio account SID, your Twilio account authentication token, and your Twilio phone number. Because the publisher is not currently checking for a notification that its message was recieved, the caching and snapshot features are unreliable in version 1.0.0.

Points Overview

Twilio published points allow you to format your information meaningfully and send it on to multiple phone numbers.

Attributes common to all published points are:

  • Name: the name of the point from its original data source
  • Status: whether publishing this point is enabled or not
  • Type: the data type of the point, inherited from the data source. Formatting treats all data as strings.
  • Format String: The format string for your message. The value of the data point will replace every '~' in the format string
  • To Phone Number: The phone number you wish to message.

Advanced Formatting

If your data is more complex, there is a second formatting option. If the point being published is of the form (val1`val2`val3) (accent, `, is the delimiter) it will be interpreted as permitting multiple replacements. A format string can have multiple replacement characters ('~') which will be replaced sequentially. For instance, if point p has format string "Chiller is at usage ~ with five-minute valve proportion ~", and were set with the value (CRITICAL`.95) the resulting text should read "Chiller is at usage CRITICAL with five-minute valve proportion .95". This means you can publish a meta point, using its capability to build your data, and perform the data transformations required to get you the information you want conveniently.
