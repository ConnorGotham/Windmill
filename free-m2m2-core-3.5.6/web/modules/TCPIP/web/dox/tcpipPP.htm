Overview

The point properties of TCP/IP data points contains information about how to receive and parse values of interest. If a point is queryable, its read command is issued every poll period. Any response received is searched by the value regex and extracted at the value index if enough groups were captured in the regex. If a point is settable the write command is issued upon setting, with the special string %VALUE% replaced by the value the point was set to. For instance, 'Hello %VALUE%' would output the popular hello when set to "World"
