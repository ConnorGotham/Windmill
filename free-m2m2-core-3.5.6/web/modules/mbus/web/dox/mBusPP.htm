MBus Data Point Configuration

Both serial and IP networks use the same point attributes to locate values.

  • Device Name - Label that is useful to group points together
  • Name - name of this point
  • Export ID (XID) - Unique id for import/export to other systems
  • Read permission - permissions for reading this point's values
  • Set permission - permissions for setting this point's value
