Overview

The MBus data source will poll MBus devices to collect sensor information.

Configuration

Every data source requires a Name, which can be any description. The Update period determines how often the MBus equipment is polled for data. Checking Quantize will cause the data source startup to be delayed so that pollings occur at a "rounded" point in time (e.g. if polling is every 10 seconds, polls will occur on the 10th second, the 20th second, etc). Otherwise, polling will begin immediately.
