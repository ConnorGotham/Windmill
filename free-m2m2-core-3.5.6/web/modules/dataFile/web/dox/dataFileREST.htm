Overview

There is a REST api with special permissions for uploading and triggering a configured data source to process one or more files. The permission is defined on the system settings page and must be supplied to the user in order for them to be allowed to process files.

REST Urls

  • /rest/v2/data-file/{xid}/import - POST files to upload as Multipart Form Data using a configured data source's xid in the path. The result is an empty body with a URL in the header and a Resource ID for status updates.
  • /rest/v2/data-file/import/{resourceId} - GET get the status of an import
