The Variable name is the name by which the point object can be referred in the script.

The Data type is the Mango representation of the value. Values provided by the set function are automatically coerced to this type where possible.

The Settable checkbox indicates whether the point is settable outside of the data source's script

The Updates context checkbox indicates whether the point triggers execution of the data source's script. You should not set values to points which execute the script from within the script.
