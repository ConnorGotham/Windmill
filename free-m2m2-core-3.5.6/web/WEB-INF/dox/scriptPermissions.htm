Script Permissions

Script Permissions are used by any Script Utility that accesses data points. The permissions are separated into 4 categories.

  • Data Source Permission - the script will inherit permissions for accessing data sources from these groups
  • Data Point Read Permission - the script will inherit permissions for reading data points from these groups
  • Data Point Set Permission - the script will inherit permissions for setting data points from these groups
  • Custom Permission - Future use for accessing module custom permissions

Script Utilities That Use Permissions

  • RuntimeManager
  • DataSourceQuery
  • DataPointQuery
