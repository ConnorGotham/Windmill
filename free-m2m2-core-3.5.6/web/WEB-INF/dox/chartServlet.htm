The chart servlet allows requesting a customizable chart image for any data point in Mango via a browser URL request. This section describes how to use this servlet.

The url for the chart servlet is: http://mangoHostName/chart/{parameters} where {parameters} are defined in this help file.

General Format

  • The URL must end in .png
  • All times are in milliseconds since Jan 1 1970 (Java Epoch)
  • All durations are in milliseconds
  • Items in brackets [ ] are optional (do not include brackets in url)
  • Multiple data points can be used by separating them with an underscore _
  • The pipe | symbol is used to separate a data point id and its chart color
  • Width and Height can be added to the image by appending a _w{pixels} and _h{pixels} such as _w500_h250.png
  • Colors can be any HTML color such as 'red' or 'light-blue' or a hex code prefixed with 0x

Duration Chart

Uses a from time and the duration from this time to define the chart period. Example:
http://mangoHostName/chart/{from timestamp}_{duration}_{data point id[|colour]}[_{data point id[|colour]} ...][_wWidth][_hHeight].png

From To Chart

Uses a from time and a to time to define the chart period. This chart uses a prepended ft_ to the front of the chart parameters.
Example:
http://mangoHostName/chart/ft_{from timestamp}_{to timestamp}_{data point id[|colour]}[_{data point id[|colour]} ...][_wWidth][_hHeight].png
