The Property Template for data points allows a way to manage and assign templates to data points. The point properties sections overriden by a Data Point Propertes Template are:

  • Point properties - Except for Units
  • Logging properties
  • Text renderer properties
  • Chart renderer properties

Check the Use Property Template check box to enable template management for the data points. Choose an appropriate template for the point from the drop down. Only templates that are compatible with the data point are displayed. When using a template the inputs that are provided by the template are disabled and you are forced to use the template's values.

Editing

To edit a template click the Edit Template button to enable the inputs for the data point. Next change any desired inputs and then click the Update Template button to save the changes into Mango. Caution this will update ALL data points that are currently using this template.

Create New Template

To create a new template click the Edit Template button. Change any desired inputs and then click the Save As New Template button. This will prompt the user for a name and the option to Save or Cancel.
