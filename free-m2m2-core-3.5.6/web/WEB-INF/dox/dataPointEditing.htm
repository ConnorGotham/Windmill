This page provides controls for editing the properties of a point. The icon provides a link to the read-only point details page; it is visible only for points that have been saved.

The data source name is provided for reference. The icon is a link to the data source editing page where the value locator configuration can be found, and is provided as a convenience.

The Point name is required, and can be any text value. This name will be used as the visual identifier for the point throughout the system.

Often, the point name is supplemented with the Device name when it is shown in lists with other points. The two together are referred to as the "extended point name". The device name is initially defaulted to the data source name, but can be changed. The two together allow a point to be uniquely identified in an instance, especially in cases where there are multiple points with the same name.

Point permissions are a comma-delimited list of virtual "group" names that are conferred the associated permission. Read permission allows the user to view the point various places, including the data point details page. Set permission allows the user to set the value of a settable point.

You can specify a Unit for a point. Type in the symbol for the unit e.g. kg, L, mA, Hz, lb. Note that for some units there are multiple ways of entering the unit, e.g. for degrees celsius you can enter °C or Celsius, for ohms you can use Ω or Ohm. Also remember that there are sometimes multiple definitions of a unit, e.g. there are metric tonnes (t), short tons (ton_us) and long tons (ton_uk). You can use all SI prefixes on SI units, e.g. you can enter W for watt and also µW (microwatt), MW (megawatt) etc.

Note that there are also some units internally defined in this system:
  • btu - British thermal unit
  • thm - Thermal Unit (BTU * 100000)
  • psi - Pounds per square inch
  • rpm - Revolutions per minute
  • ppm - Part per million
  • ppb - Part per billion
There are also some aliases for common units defined here:
  • Celsius - °C
  • Fahrenheit - °F

Units can also be constructed as an expression of units, such as expressing the units of acceleration m/s^2 or a unit of surface loss m^2/s. Kilowatt hours is special: while kW*h would be the means of building it, kWh is the standard suffix and both are supported units. Similarly, gal/min is more simply labeled gpm in common practice, and both are accepted units.

The Displayed unit specifies which unit the renderer should convert to for display (if supported by the renderer). Input format is the same as for Unit.

The Integral unit used in the display of statistic and specifies which unit the integrated total should be displayed in. Input format is the same as for Unit.

The Chart color is the default color of the point's line in images charts of multiple points. Color values can be an HTML-style hex, or any color name in the HTML specification. For example, "#2468AC", "red", and "dodgerblue" are all valid values.

The point can influence its default charting through a Rollup which will plot that statistical metric when a rollup of 'Point default' is requested. A rollup has a Rollup period and a Rollup period type.

Both rollups and raw data sets can also be put through the Simplify algorithm. Data sets can aspire to have a certain resulting number of values, or provide a magnitude for what is significant. While this can take more resources and time to get the data set, the data set should have its minor variances simplified. A point set to simplify either to a target or a tolerance it will use the Simplify argument for that value.

Data points can restrict outgoing set values with the Prevent extreme sets option. This allows the admin to specify an upper and lower bound for a point's values, such that an operator cannot set a value out to the device that is either below the Low set limit or above the High set limit.
